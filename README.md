# Sales Performance Analytics Dashboard

## Overview

The Sales Performance Analytics Dashboard is a Tableau-based project developed during my internship to analyze retail sales data and present key business information through interactive visualizations. The dashboard helps users explore sales trends, profit, quantity, customer segments, product categories, and regional performance in an organized and user-friendly manner.

---

## Project Objectives

* Analyze retail sales data using Tableau.
* Build interactive dashboards for business reporting.
* Compare sales and profit across different categories and regions.
* Study customer segments and shipping methods.
* Present data in a visual format to support business decisions.

---

## Key Features

* Interactive Tableau dashboard
* Sales analysis by category
* Sub-category performance comparison
* Profit and quantity analysis
* Customer segment insights
* Ship mode comparison
* State-wise sales visualization
* Dynamic filters for data exploration

## Dataset Information

This project uses two datasets.

### Retail Store Order Details

* Order ID
* Order Date
* Ship Date
* Customer Details
* Sales
* Quantity
* Discount
* Profit
* Region
* State

### Retail Store Product Details

* Product ID
* Category
* Sub-Category
* Product Name

The datasets are connected using the **Product ID** field to generate complete business insights.

---

## Dashboard Components

The dashboard includes:

* Sales by Category
* Sales by Sub-Category
* Profit Analysis
* Quantity Analysis
* Customer Segment Distribution
* Ship Mode Analysis
* State-wise Sales Comparison

Interactive filters allow users to explore different views of the data.

## How to Use

1. Clone or download this repository.
2. Open `SUPER STORE INTERACTIVE DASHBOARD.twbx` using Tableau Desktop or Tableau Public.
3. Explore the dashboard with the available filters.
4. Refer to the presentation for an overview of the project.
