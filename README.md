# Data Analysis Using Tableau

## Overview

This repository contains my internship project on **Data Analysis Using Tableau**, completed as part of the **Data Science + AI/ML Modeling + BigQuery Internship**. The project focuses on transforming raw datasets into interactive dashboards that provide meaningful business insights through data visualization.

The repository includes multiple Tableau dashboards, datasets, project documentation, and presentation files developed during the internship.

---

## Project Objectives

* Learn the fundamentals of Tableau and business intelligence.
* Analyze real-world datasets using data visualization techniques.
* Build interactive dashboards for effective decision-making.
* Present business insights through charts, filters, and visual reports.

---

## Dashboards Included

### 1. Netflix Insights Dashboard

* Analysis of titles based on ratings.
* Country-wise title distribution.
* Movies and TV shows comparison.
* Year-wise and monthly trends.

### 2. HR Insights Dashboard

* Employee distribution by department.
* Hiring and termination trends.
* Gender analysis.
* Age distribution.
* Top job roles.

### 3. Wine Sales Insights Dashboard

* Revenue analysis by region.
* Top-performing customer segments.
* Revenue comparison across locations.
* Customer order analysis.

### 4. Super Store Data Insights Dashboard

* Sales and profit analysis.
* Forecast visualization.
* Product and customer performance.
* Distribution of sales using treemaps.

### 5. Super Store Interactive Dashboard

* Sales analysis by category and sub-category.
* Quantity and profit visualization.
* Segment-wise performance.
* Ship mode analysis.
* State-wise sales comparison with interactive filters.
